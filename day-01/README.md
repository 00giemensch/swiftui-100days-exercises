# Day 01 — Variables, constants, strings, numbers

## Exercises

**1a.** Создай константу с названием своего любимого фильма. Выведи её длину.

**1b.** Создай переменную `city` со значением (придумай свое). Выведи ее в верхнем регистре.

**1c.** Создай константу `websiteURL` со значением "https://apple.com". Проверь, начинается ли она с "https" и заканчивается ли на ".com". Выведи оба результата.

**1d.** Создай переменную `lives = 3`. Игрок умер — забери жизнь. Игрок нашёл бонус — добавь 2. Выведи итог.

**1e.** Создай константу `population = 8_000_000_000`. Выведи её. Напиши в комментарии что означает это число.

**1f.** Создай переменную `temperature = 36.6`. Пациенту стало хуже — температура выросла на 1.2. Выведи итог.

**1g.** Создай константу `songTitle` со значением "Bohemian Rhapsody". Выведи количество символов и проверь, заканчивается ли на "dy".

**1h.** Создай переменную `bankBalance = 1_000`. Купи что-то за 350, получи зарплату 2_500, заплати аренду 800. Выведи итоговый баланс.

**1i.** Создай константу `year = 2024`. Проверь, делится ли год на 4 (високосный ли). Выведи результат.

**1j.** Создай многострочную строку — три строки из меню кафе (блюдо, цена, описание). Выведи её.

## 🏆 Challenge — Hero Profile

**1k.** Константа `heroName` — придумай крутое имя героя.

**1l.** Переменная `health = 100`. Герой получил урон 35, нашёл аптечку +20, попал в ловушку -15. Выведи итоговое здоровье.

**1m.** Проверь: делится ли итоговое здоровье на 5.

**1n.** Выведи `heroName` заглавными буквами — как будто его объявляют на арене.

**1o.** Многострочная строка `heroDescription` — имя, класс, оружие. Выведи её длину.
