# Day 03 — Arrays, Dictionaries, Sets, Enums

## Exercises — Arrays

**3a.** Создай массив `groceries` из 4 продуктов. Добавь в конец ещё один через `.append()`. Выведи весь массив.

**3b.** Создай массив `colors` из 5 цветов. Выведи первый и последний элемент по индексу.

**3c.** Создай массив `tasks` из 4 задач на день. Удали вторую задачу через `remove(at:)`. Выведи количество оставшихся задач через `.count`.

**3d.** Создай массив `players` из 6 имён. Проверь через `.contains()`, есть ли там определённое имя (придумай два варианта — которое есть и которого нет).

**3e.** Создай массив чисел `prices` (5 значений). Выведи отсортированную версию через `.sorted()`, не изменяя оригинальный массив.

**3f.** Создай массив `queue` из 4 имён людей в очереди. Выведи массив в обратном порядке через `.reversed()`.

**3g.** Создай пустой массив `Array<String>` с именем `playlist`. Добавь туда 4 названия песен через `.append()`. Выведи итоговый плейлист.

## Exercises — Dictionaries

**3h.** Создай словарь `capital` где ключи — названия трёх стран, а значения — их столицы. Выведи столицу одной из стран.

**3i.** Создай словарь `student` с ключами `"name"`, `"grade"`, `"school"`. Выведи все три значения через default value `"Unknown"`.

**3j.** Создай словарь `inventory` (тип `[String: Int]`) — 3 товара и их количество на складе. Выведи количество товара, которого там нет, используя default значение `0`.

**3k.** Создай пустой словарь `[String: String]` с именем `passwords`. Добавь туда 3 пары логин-пароль. Перезапиши пароль для одного из логинов. Выведи итоговый словарь.

**3l.** Создай словарь `scoreByPlayer` (тип `[String: Int]`) с 4 игроками. Выведи количество игроков через `.count`.

## Exercises — Sets

**3m.** Создай множество `uniqueVisitors` из массива с повторяющимися именами (минимум 2 дубликата). Выведи множество и посчитай сколько там реально элементов.

**3n.** Создай пустое множество `Set<String>`. Добавь туда 5 элементов через `.insert()`, два из которых одинаковые. Выведи `.count` — объясни в комментарии почему он меньше пяти.

**3o.** Создай множество `bannedWords` из 5 слов. Проверь через `.contains()` есть ли в нём конкретное слово.

## Exercises — Enums

**3p.** Создай enum `TrafficLight` с тремя case: `red`, `yellow`, `green`. Создай переменную, присвой ей `.red`, затем измени на `.green`.

**3q.** Создай enum `Direction` с case `north`, `south`, `east`, `west`. Напиши переменную текущего направления персонажа и измени его три раза.

**3r.** Создай enum `OrderStatus` с case `pending`, `shipped`, `delivered`, `cancelled`. Создай переменную для статуса заказа и проведи его через все стадии кроме `cancelled`.

## 🏆 Challenge — Inventory System

Ты разрабатываешь простую систему учёта для интернет-магазина.

**3s.** Создай enum `Category` с case `electronics`, `clothing`, `food`, `books`.

**3t.** Создай массив `productNames` из 5 товаров.

**3u.** Создай словарь `stock` (тип `[String: Int]`), где ключи — это товары из `productNames`, а значения — количество на складе.

**3v.** Создай множество `outOfStockAlerts` — добавь туда названия товаров, у которых на складе `0` (выбери вручную 1-2 товара с нулевым остатком).

**3w.** Выведи через string interpolation отчёт по каждому товару в формате: `"Товар: Наушники, на складе: 5"`, используя default значение для несуществующих ключей.

**3x.** Проверь через `.contains()`, есть ли первый товар из `productNames` в `outOfStockAlerts`, и выведи результат.
