# Day 04 — Type annotations

## Exercises

**4a.** Создай константу `bookTitle` с явной type annotation `: String`. Присвой значение — название любимой книги.

**4b.** Создай переменную `temperature` с type annotation `: Double`, но присвой ей целое число (например `20`). Выведи и проверь — какого типа получилось значение.

**4c.** Создай переменную `attempts: Int`, не присваивая значения сразу. Чуть ниже (как будто после "сложной логики") присвой ей значение `3`. Выведи.

**4d.** Создай пустой массив `favoriteGenres` с type annotation `: [String]`. Добавь туда 3 жанра через `.append()`.

**4e.** Создай пустой словарь `userSettings` с type annotation `: [String: Bool]`. Добавь туда 3 настройки (например `"darkMode"`, `"notifications"`) со значениями `true`/`false`.

**4f.** Создай пустое множество `Set<Int>` с именем `rolledDice`. Добавь туда 6 чисел через `.insert()`, среди которых есть повторы (имитация бросков кубика). Выведи `.count` — будет меньше шести из-за дубликатов.

**4g.** Создай enum `Weather` с case `sunny`, `rainy`, `cloudy`, `snowy`. Создай константу `todayWeather: Weather` и присвой одно из значений.

**4h.** Попробуй создать константу `let total: Int = "5"` (со строкой вместо числа) — посмотри на ошибку компилятора. Исправь её на правильный тип и опиши в комментарии своими словами, что произошло.

## 🏆 Challenge — Checkpoint 2 (по заданию Пола)

Это задание Paul Hudson даёт прямо в теории — вот расширенная версия для практики:

**4i.** Создай массив строк `watchedMovies` минимум с 8 элементами, среди которых **есть повторяющиеся** названия (ты несколько раз смотрел один и тот же фильм).

**4j.** Выведи через `.count`, сколько всего фильмов в списке (включая повторы).

**4k.** Преврати массив в `Set`, используя `Set(watchedMovies)`, и выведи количество **уникальных** фильмов.

**4l.** Выведи через string interpolation итог в формате: `"Всего просмотров: 8, уникальных фильмов: 5"`.

**4m.** Добавь к изначальному множеству ещё один фильм, которого там не было, через `.insert()`. Проверь через `.contains()`, что он теперь там есть.
